# Portfolio Website Structure - Carlos Torres

## File Organization

### Core Files
- **index.html** - Homepage with hero section and project overview
- **projects.html** - Detailed project showcase with filtering
- **skills.html** - Interactive skills visualization and certifications
- **contact.html** - Contact form and professional links

### Assets Directory
- **resources/** - Images, icons, and media files
  - **hero-bg.jpg** - Hero background image
  - **project-images/** - Project screenshots and visuals
  - **skill-icons/** - Technology and framework icons
  - **certifications/** - Certificate images and badges

### JavaScript
- **main.js** - Core functionality and interactions

## Page Structure

### Homepage (index.html)
1. **Navigation Bar**
   - Logo/Name with animated treatment
   - Menu links: Home, Projects, Skills, Contact
   - Mobile hamburger menu

2. **Hero Section**
   - Animated background with particle system
   - Large typography with typewriter effect
   - Professional tagline and key skills
   - Call-to-action button to projects

3. **Skills Overview**
   - Interactive radar chart showing competencies
   - Animated counters for key metrics
   - Technology stack visualization

4. **Featured Projects**
   - Horizontal scrolling carousel
   - Project cards with hover effects
   - Quick stats and technology tags

5. **Experience Timeline**
   - Interactive timeline with key milestones
   - Hover effects revealing details
   - Connection to skills and projects

6. **Footer**
   - Copyright information
   - Social media links
   - Contact information

### Projects Page (projects.html)
1. **Filter Navigation**
   - Category filters: ML, Data Viz, Analysis, Web Apps
   - Technology filters: Python, TensorFlow, Tableau, etc.
   - Search functionality

2. **Project Grid**
   - Responsive card layout
   - Hover animations revealing details
   - Technology badges and metrics

3. **Project Details Modal**
   - Full-screen overlay with project information
   - Image gallery and code snippets
   - Live demo links and GitHub repository

### Skills Page (skills.html)
1. **Interactive Skills Matrix**
   - Heatmap showing proficiency levels

   - Animated progress indicators

2. **Certification Showcase**
   - Visual timeline of certifications
   - Hover effects showing completion dates
   - Links to verification sources

3. **Learning Progress**

4. **Technology Ecosystem**
   - Interactive diagram showing tool relationships

### Contact Page (contact.html)
1. **Contact Form**
   - Clean form design with validation

2. **Professional Links**
   - LinkedIn, GitHub, Email
